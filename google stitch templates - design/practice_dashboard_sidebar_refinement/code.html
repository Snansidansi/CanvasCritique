Spencerian Exercise: Upper Loops

Spencerian script is defined by its elegance and rhythmic flow. In this exercise, we focus on the **upper loops** found in letters like l, h, k, b, and f.

Instructions:

  1. Start at the baseline with a light upward stroke.
  2. Maintain a consistent 55-degree slant throughout the entire loop.
  3. The peak of the loop should touch the ascender line gently.
  4. The downward stroke should be slightly thicker (a "swell") as it returns to the baseline.
  5. The cross-over point should be exactly at the header line.
